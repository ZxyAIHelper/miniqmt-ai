# 目标策略记录

## 资金与目标

- 初始资金：10000 元
- 策略目标：稳定赚钱，优先控制回撤和亏损时间，不追求短期暴利
- 适用范围：MiniQMT / xtquant 外部 Python API 可转债回测
- 当前用途：作为寻找、筛选和回测可转债策略时的指标标准

## 核心指标门槛

| 指标 | 目标 | 说明 |
| --- | --- | --- |
| 年化收益率 | > 8% | 先追求稳健正收益，不优先追求高收益 |
| 最大回撤 | < 10%，最多不超过 15% | 10000 元资金回撤 10% 即亏损 1000 元，需要重点控制 |
| Calmar 比率 | > 1，最好 > 1.5 | 年化收益率 / 最大回撤，越高代表收益和风险越平衡 |
| 月度胜率 | > 60% | 多数月份应尽量保持正收益 |
| 最长回撤时间 | < 6 个月 | 回本时间过长会影响策略可坚持性 |
| 单月最大亏损 | < 5% | 避免单月出现明显大亏 |
| 夏普比率 | > 1，最好 > 1.5 | 衡量收益相对波动是否合理 |
| 成本后收益 | 必须为正 | 回测必须考虑手续费、滑点等交易成本 |
| 样本外表现 | 必须为正或明显稳健 | 防止只在历史数据中过拟合 |

## 可转债策略筛选条件

| 指标 | 建议 |
| --- | --- |
| 持仓数量 | 3 到 8 只 |
| 单券仓位 | 单只不超过 30% 到 40% |
| 转债价格 | 优先关注 100 到 130 区间 |
| 溢价率 | 不宜过高，优先考虑双低逻辑 |
| 剩余规模 | 避免规模过小、流动性差的转债 |
| 日成交额 | 优先选择成交活跃的转债 |
| 强赎风险 | 必须过滤或单独处理 |
| 信用风险 | 避免明显评级过低或异常标的 |

## 优先寻找的策略方向

1. 双低策略：综合考虑转债价格和转股溢价率，适合作为稳健策略起点。
2. 低价策略：优先选择价格较低的转债，防守属性更强。
3. 低价 + 流动性过滤：适合 10000 元小资金，减少成交困难和滑点影响。
4. 双低 + 强赎过滤：避免持有临近强赎或风险较高的转债。
5. 定期轮动策略：按周或按月调仓，选择排名靠前的转债。

## 回测时必须检查

- 是否加入手续费和滑点。
- 是否统计最大回撤、Calmar、夏普、月度收益、年度收益。
- 是否统计最长回撤时间。
- 是否输出每笔买入和卖出记录。
- 是否检查分年度表现，避免收益只来自单一年份。
- 是否做样本外测试，例如用早期数据调参，用后期数据验证。
- 是否检查参数敏感性，避免参数稍微变化后策略失效。

## 当前策略选择倾向

优先寻找：

```text
双低 + 流动性过滤 + 强赎过滤 + 控制最大回撤
```

暂不优先寻找：

```text
高频交易策略
单一高收益参数策略
高价转债追涨策略
未考虑手续费和滑点的策略
依赖极端行情或单一年份收益的策略
```
